# Coffee Lea Landing Page

A React + Vite landing page built with Tailwind CSS and GSAP animations.

## Features

- Responsive coffee shop landing page layout
- Tailwind CSS styling
- GSAP entry animations
- Lucide icons

## Scripts

- `npm install` — install dependencies
- `npm run dev` — run the development server
- `npm run build` — build for production

## Notes

The page uses placeholder imagery from remote sources. Replace image URLs with your own assets as desired.
